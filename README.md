# Wildfire Satellite Mission Analysis

## Overview

This project investigates whether a low Earth orbit satellite could effectively monitor wildfires from space.

The goal is to explore tradeoffs between orbital altitude, coverage, revisit time, and the number of satellites required for useful wildfire detection.

This project is being developed as a personal aerospace engineering portfolio project.
